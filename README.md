# PyMOL structural analysis
This repository contains structural visualization and qualitative molecular analysis workflows developed using PyMOL.

## Overview
The project focuses on clean and minimal structural representations of biomolecular systems relevant to computational chemistry and structure-based drug discovery. The visualizations emphasize structural organization, molecular interactions and qualitative inspection of RNA architectures.

## Selected structure
The structural visualizations in this project were generated using the RNA aptamer structure available in the Protein Data Bank under the accession code 1EHZ. The structure was selected as an example of RNA architecture relevant to structural biology and computational analysis workflows.
