# PyMOL structural analysis
This repository contains structural visualization and qualitative molecular analysis workflows developed using PyMOL.

## Overview
The project focuses on clean and minimal structural representations of biomolecular systems relevant to computational chemistry and structure-based drug discovery.

The visualizations emphasize structural organization, molecular interactions and qualitative inspection of RNA architectures.

## Contents
- RNA structural visualization
- Cartoon representations
- Surface representations
- Structural detail inspection
- Publication-style molecular rendering

## Tools
- PyMOL
- Protein Data Bank (PDB)

## Concepts explored
- Structural biology
- Molecular visualization
- RNA architecture
- Biomolecular representation
- Computational chemistry workflows
